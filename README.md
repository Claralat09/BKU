Ich habe den Code in Visual Studio Code für die Playground-Platine der BWS geschrieben. Die meisten Programme beginnen mit dem Einbinden der benötigten Bibliotheken, damit die Bauteile wie Servo, Ultraschallsensor, Joystick oder das OLED-Display verwendet werden können. In der setup()-Funktion werden alle Bauteile gestartet. Danach wird die loop()-Funktion ständig wiederholt. Dort werden die Sensoren ausgelesen und mit if-Bedingungen überprüft. Je nach Wert führt das Programm eine Aktion aus, zum Beispiel den Servo bewegen oder etwas auf dem OLED-Display anzeigen.

Am längsten bin ich beim Joystick stecken geblieben. Ich musste zuerst herausfinden, welche Werte die X- und Y-Achse zurückgeben und wie ich diese im Code verwenden kann. Um das besser zu verstehen, habe ich die Werte auf dem OLED-Display ausgegeben. So konnte ich sehen, welche Werte bei den verschiedenen Bewegungen entstehen und den Code richtig anpassen.


In diesem Projekt habe ich die wichtigsten Funktionen eines Programms für die BWS-Playground-Platine kennengelernt. Ich habe verstanden, dass setup() für die Initialisierung der Bauteile zuständig ist und loop() den Programmablauf ständig wiederholt. Ausserdem habe ich gelernt, den Joystick und den Ultraschallsensor auszulesen, Werte auf dem OLED-Display anzuzeigen und den Servomotor mit if-Bedingungen zu steuern. Dadurch habe ich einen besseren Einblick in die Programmierung mit Visual Studio Code und die Ansteuerung der verschiedenen Bauteile erhalten.

In diesem Unterricht habe ich gelernt, wie man verschiedene Bauteile wie den Servo, den Ultraschallsensor, den Joystick und das OLED-Display mit der BWS-Playground-Platine programmiert. Dabei habe ich besser verstanden, wie ein Programm aufgebaut ist und wie setup(), loop() und if-Bedingungen funktionieren.
